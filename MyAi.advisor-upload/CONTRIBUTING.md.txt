# Want to Help Improve iQvault.ai?

We welcome:
- Bug reports
- Feature requests
- Code improvements
- Support for new file types (Excel, email, audio)

## How to Contribute
1. Fork the repo
2. Create a branch: `git checkout -b feature/excel-support`
3. Commit your changes
4. Push: `git push origin feature/excel-support`
5. Open a Pull Request

Let’s build the future of private AI — together.
